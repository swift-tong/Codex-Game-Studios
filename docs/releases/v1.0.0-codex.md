# Release Notes: v1.0.0-codex

## Summary

This release establishes Codex Game Studios as a Codex-first standalone project.

## Highlights

- Added root and nested `AGENTS.md` instruction files
- Promoted `.agents/` to the active skills, docs, rules, and role-definition tree
- Rewrote public repository docs for Codex usage
- Updated GitHub issue, PR, and ownership templates
- Removed archived pre-Codex assets from the public repository

## Breaking / Structural Changes

- The active instruction surface is now `AGENTS.md` plus `.agents/`
- Older non-Codex instruction files were removed from the project surface

## Upgrade Notes

- Existing users should start from `AGENTS.md`
- Update any local references from `.claude/` to `.agents/`
- Review `UPGRADING.md` for merge guidance if you are carrying local customizations

## Recommended Next Steps After Release
