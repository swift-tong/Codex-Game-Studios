# `.agents` Directory

This is the Codex-native workspace layer for Codex Game Studios.

- `agents/`: role definitions and reusable prompts
- `docs/`: workflow docs, templates, and reference material
- `rules/`: path-scoped standards
- `skills/`: reusable skills discovered by Codex

When adapting this repository for a new game, treat `.agents/` as the primary instruction tree.
